# Project MUN - Business Portfolio Website

A modern, responsive business portfolio website for Project MUN - a web development team composed of Ruben Albao, Kristian Gomez, Jonelle Mayari, and Alvin Panganiban.

## Features

- **Modern Design**: Navy blue theme with glassmorphism effects, gradients, and smooth animations
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices
- **Dark Mode**: Default dark theme with optional light mode toggle
- **Smooth Animations**: Scroll reveal animations, hover effects, and transitions
- **Sticky Navigation**: Fixed navbar with smooth scrolling to sections
- **Contact Form**: Interactive contact form with validation
- **Portfolio Gallery**: Filterable project showcase with category filtering

## Tech Stack

- **React.js** - Component-based architecture
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Modern icon library
- **Framer Motion** - Animation library (ready for integration)

## Website Sections

1. **Hero** - Project title, tagline, CTAs, and animated background
2. **About** - Team introduction with member cards and values
3. **Services** - Web development, UI/UX design, system development offerings
4. **Portfolio** - Project showcase with filtering capabilities
5. **Technologies** - Tech stack display with icons
6. **Contact** - Contact form and social links
7. **Footer** - Navigation, team credits, copyright

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

### Building for Production

```bash
npm run build
```
